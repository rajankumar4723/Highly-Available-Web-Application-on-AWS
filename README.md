# 🚀 Highly Available Web Application on AWS

## 📌 Project Overview

This project demonstrates how to build a highly available and scalable web application using AWS cloud services.

---

## 🧱 Architecture

* Created custom VPC with public and private subnets
* Deployed EC2 instances in multiple Availability Zones
* Configured Application Load Balancer for traffic distribution
* Implemented Auto Scaling for dynamic scaling
* Integrated Amazon RDS in private subnet for secure database

---

## ⚙️ Services Used

* EC2 (Elastic Compute Cloud)
* VPC (Virtual Private Cloud)
* Application Load Balancer
* Auto Scaling
* RDS (Relational Database Service)
* IAM (Identity and Access Management)

---

## 🔧 Steps Performed

1. Created VPC and subnets
2. Launched EC2 instances and installed Apache
3. Configured Load Balancer and Target Group
4. Tested application using Load Balancer DNS
5. Implemented Auto Scaling Group
6. Created RDS database in private subnet
7. Connected EC2 to RDS
